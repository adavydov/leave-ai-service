# MVP-матрица проверок заявления на отпуск (ТК РФ)

Документ фиксирует минимальные правила комплаенса для MVP.

## Границы MVP
- Проверяем формальные реквизиты заявления, даты, длительность и базовые юридические подсказки.
- Не делаем полноценную правовую экспертизу индивидуальных кейсов.
- OCR-неопределенности выводим отдельно как технические подсказки.

|---|---|---:|---|---|

| DOC-REQ-003 | missing_manager_name | warn | Указан адресат | Локальные регламенты |
| DOC-REQ-004 | missing_request_date | error | Есть дата заявления | Фиксация волеизъявления |
| DOC-REQ-005 | missing_leave_start_date | error | Есть дата начала | Определенность периода |
| DOC-REQ-006 | missing_leave_end_date | error | Есть дата окончания | Определенность периода |
| DOC-SIGN-001 | missing_signature | error | Есть подпись работника | Кадровая практика |

## Post-MVP (не входит в текущую реализацию)
- Автоматическая сверка с графиком отпусков и остатками.
- Проверка специальных оснований по подтверждающим документам.
- Организационно-специфические SLA и маршруты согласования.
